**学生管理系统**
项目简介
学生管理系统是一个基于 Java Swing 开发的桌面应用程序，旨在实现学校学生、教师、课程及成绩的综合管理。系统支持多角色登录（管理员、教师、学生），并根据不同角色提供相应的功能权限，实现了数据的增删改查及业务逻辑管理。
**功能模块**
1. 用户登录模块
   支持管理员、教师、学生三种角色登录
   角色切换功能（支持右键快捷切换）
   输入验证及登录状态管理
   登录成功后根据角色分配权限
2. 个人中心模块
   查看个人基本信息
   编辑个人资料（根据角色显示不同字段）
   密码修改功能
   退出登录功能
3. 学生管理模块（仅管理员）
   查看所有学生信息
   新增学生（包含基本信息录入及验证）
   编辑学生信息
   删除学生记录
   数据刷新功能
4. 教师管理模块（仅管理员）
   教师信息表格展示（教师 ID、用户名、性别、职称、年龄等）
   教师信息管理功能（增删改查）
5. 课程管理模块
   管理员 / 学生：查看所有课程信息
   教师：查看自己的授课课程
   学生：查看已选课程及退课功能
   课程信息维护（名称、学分、课时、描述、授课教师等）
6. 成绩管理模块
   教师查看选课学生列表
   成绩录入与批改功能

**技术栈**
   编程语言：Java
   UI 框架：Swing
   数据库：Apache Derby（嵌入式数据库）
   构建工具：IDE 内置编译工具
   测试框架：JUnit
   环境要求
   JDK 版本：17 及以上
   开发工具：IntelliJ IDEA（推荐）或其他 Java IDE
   操作系统：Windows/macOS/Linux
